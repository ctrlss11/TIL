## 유용한 내장 함수
* help() : 도움말
  * class, 함수 등의 사용법을 보고 싶을 때
* dir() : 현재 메모리에 할당된 변수들 확인
  * class, 함수를 넣으면 그 안에 정의된 함수를 확인
* del() : 삭제
```python
help(help)
dir()
dir(list)          
```
